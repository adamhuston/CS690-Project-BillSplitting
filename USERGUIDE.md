User Guide
Features (UC1: Record a Debt)
Record a debt: track money someone owes you
Enter the debtor, amount, date, and optional description
Date defaults to today if left blank
Each debt gets a unique ID for future reference
All amounts are in USD
Use
Run the application (see Deployment instructions)
Enter the person ID
Enter the amount owed
Enter a date or press Enter for today
Optionally, enter a description
The system confirms the debt was recorded
Known limitations
Data does not persist
Person IDs are not validated - you can enter any positive integer
Only one currency (USD) is supported
No way to list saved debts yet - you can only see the confirmation of the one you just entered
