# CS690-Project-BillSplitting
CS690 Project

A console app for tracking shared debts and payments. Built with .NET 9

See the [WIKI](../../wiki) for user guide, deployment, development, and other documentation.

## Features
UC1 (built): Create a debt

UC2 (planned): Record a payment

UC3 (planned): Record a bill

UC4 (planned): View summaries of transactions

## Quick Start
```
dotnet build
dotnet run --project BillSplittingUI
```
